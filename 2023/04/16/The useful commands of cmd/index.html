Network

You can get all information about your devices’ network via the command “netsh”

if you want know more about ‘netsh’ via Network Shell (Netsh) | Microsoft Learn

if you wang to know the SSID of Wlan that you connected , you can use this command:

netsh wlan show interfaces | findstr /c:”SSID”

if you want to find your ipv4 address ,you can just use this command:

netsh interface ipv4 show ipaddress interface
